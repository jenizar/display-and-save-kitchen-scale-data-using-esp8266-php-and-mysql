# display-and-save-kitchen-scale-data-using-esp8266-php-and-mysql
display and save kitchen scale data using esp8266, php, and mysql

materials:

digital kitchen scale custom, esp8266, HX711 load cell amplifier, psu 5v/3A, jumper cable.
